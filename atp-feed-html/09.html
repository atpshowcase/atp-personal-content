#7
Kontrak sama Masa Depan!

Indexes: Masa Depan Lu Bakal Thank You

⚡ Apa yang di-Index:
• Foreign keys
• WHERE clause columns
• JOIN columns
• ORDER BY columns

🚫 Apa yang di-Avoid:
• Low cardinality (boolean flags)
• Columns yang jarang di-query
• Too many indexes pada write-heavy tables

Index slow down writes! Pake EXPLAIN ANALYZE!

#9
Design, Bukan Tuning!

Performance = Pilihan Design

🚫 Salah Approach:
• Weeks tuning query plans
• Adding cache everywhere
• Chasing connection pools

✅ Bener Approach:
• Move frequently-joined column ke main table
• Split bloated table (hot/cold)
• Add covering index
• Partition by time range

Single schema redesign = fix selamanya!
